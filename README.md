# Определитель частоты голоса

#### Особенности:
- запускается в Docker
- основной проект реализован как Asp Core WebApi, включая принятые практики: внедрение зависимостей, использованием контроллеров, использование swagger
- коммуникация с клиентской частью идет через веб-сокет
- используется потоковая обработка данных с клиентской стороны
- поток данных берется из микрофона пользователя
- кдючевая логика определения частоты голоса находится в классе PitchDetector

#### Реализовать на будущее:

- поддержка БД
